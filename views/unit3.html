Unit 3: Algorithms

In Unit 3, different algorithms were discussed and used in applications of C.

Linear Search

Linear Search - Moving One by One

LINEAR SEARCH LINK

Linear search is used to find an element in a list. It is the most basic sorting algorithm, which means it will also take the most amount of time.

Bubble Sort

Bubbling Up to the Top

BUBBLESORT LINK

Bubblesort is a searching algorithm that can be used to sort everything in order. If a number is less than the next number, it will swap positions until it is less than the other. It repeats until the process cannot happen anymore.

Selection Sort

Swap, swap, swap

SELECTION SORT LINK

Selection sort looks at a element in a list and compares it to the lowest number in the list. If it is behind that number, it swaps and changes places. This swapping repeats until it cannot be swapped anymore.

Insertion Sort

INSERTION SORT LINK

Insertion sort splits up the elements into a seperated and unseperated array. It compares the unseperated array with seperated array and switches it until it is the further most in the array.

Binary Search

BINARY SEARCH LINK

In binary search, you divide your search of elements in half. Each time if the element cannot be found, you divide the space in half and repeat until the element is found or sorted.
